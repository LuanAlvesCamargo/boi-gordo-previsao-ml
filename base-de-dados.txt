BDMEP: Contém dados digitais de estações meteorológicas convencionais e automáticas do Brasil, com milhões de informações de séries históricas. Os dados são públicos e podem ser acessados através do portal.

Serviços no Portal: O portal oferece acesso a dados em tempo real e históricos, incluindo mapas interativos e a possibilidade de download dos dados. É possível solicitar dados históricos por e-mail após a escolha da estação e dos parâmetros desejados.

API: O INMET disponibiliza acesso facilitado via API (Application Programming Interface) para integrar dados meteorológicos.
